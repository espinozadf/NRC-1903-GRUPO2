*Back-End :
- Carpeta : control-acceso-unab-backend

- Descripción : Carpeta donde se encuentra el proyecto back-end de la 
aplicación móvil, en este backend se desarrollarón y publicaron las    API's que son consumidas por el front-end de la aplicación móvil.

- Programación : Lenguaje de programación PHP 8.1 con framework Laravel 10

*Front-End:
- Carpeta : control-acceso-unab-frontend

- Descripción : Carpeta donde se encuentra el proyecto front-end de la 
aplicación móvil, en este frontend se desarrolló el entorno visual de la aplicación móvil, las interfaces y toda la lógica por el lado del cliente.

- Programación : Lenguaje de programación Typescript con framework Ionic 7.1 con Angular
